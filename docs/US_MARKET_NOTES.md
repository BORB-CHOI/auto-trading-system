# 미국 주식 시장으로 갈 때 — 변화점 정리

> 본 문서는 메인 지침서(PROJECT_GUIDELINES.md)의 **차이점만** 담는다.
> 핵심 4-layer 구조와 원칙은 동일하다.

---

## TL;DR — 변하는 것 / 변하지 않는 것

| | 한국 시장 | 미국 시장 |
|---|---|---|
| 4-layer 구조 | 동일 | 동일 |
| LLM 역할 (Layer 2까지) | 동일 | 동일 |
| 학습 모델 미루기 | 동일 | 동일 |
| 데이터 3분할 | 동일 | 동일 |
| Regime-adaptive risk | 동일 | 동일 |
| **데이터 소스** | pykrx, KIS, DART | yfinance, Polygon, IEX |
| **수급 데이터** | 일별 외인/기관/개인 ⭕ | 분기 13F만 ❌ |
| **세금** | 22% (대주주만) | 양도소득세 22% (250만원 공제) |
| **수수료** | 0.015~0.25% | 0.0~0.1% (Robinhood 등 무료) |
| **거래시간** | 한국 주간 09:00~15:30 | **밤 23:30~06:00** (서머타임 22:30) |
| **유동성** | KOSPI 대형주 OK, KOSDAQ 일부 | 압도적으로 풍부 |
| **공시 빠르기** | 분기 / 사후 / 정성적 | 분기 의무 / 8-K / 가이던스 / **earnings call 녹취** |
| **언어** | 한국어 (LLM 살짝 약함) | 영어 (LLM 가장 강함) |
| **차별화 가능성** | 높음 (틈새 많음) | **낮음** (이미 다 함) |

## 1. 가장 큰 함정 — 본인 edge가 사라진다

한국 시장의 **본인이 가질 수 있던 edge들**:
- 한국어 NLP — 영어권 회사가 안 들어옴
- 일별 외인/기관 수급 — 미국은 분기 13F뿐
- 한국 유튜브 시황 생태계 — 영어권에 없음
- 1,000만원 시드 — 헤지펀드가 못 다루는 사이즈

미국 시장 가면 **위 4개 모두 사라지거나 약해진다.**

미국 시장은 **세계에서 가장 효율적인 시장**이고, 영어 NLP·sentiment trading은 르네상스·시타델·투시그마·점프트레이딩·진스트리트가 **15년간 수십억 달러 인프라로** 돌리고 있다. 본인이 GPT API로 헤드라인 분류해서 그 위에 올라타는 건 매우 어렵다.

**거꾸로 말하면:** 한국에서 작동하는 시스템을 미국으로 옮기면 안 된다. 한국 시장의 비효율을 노리는 시스템이기 때문이다.

## 2. 데이터 측면 변화

### 2.1. 무료 데이터 소스

| 데이터 | 한국 | 미국 |
|---|---|---|
| 가격 / 거래량 | pykrx | **yfinance** (yahoo finance), pandas-datareader |
| 일별 수급 | pykrx (외인/기관/개인) | ❌ 없음. 13F는 분기 |
| 공시 | DART OpenAPI | **SEC EDGAR API** (무료, 풍부) |
| 뉴스 헤드라인 | 네이버/다음 (스크래핑) | Yahoo Finance, MarketWatch (스크래핑), GDELT |
| 실적 발표 | DART, 네이버 금융 | **Earnings Whispers**, Seeking Alpha |
| Earnings call | ❌ 텍스트 거의 없음 | **Seeking Alpha**, Motley Fool 트랜스크립트 |
| 옵션 데이터 | KOSPI200 일부 | **압도적으로 풍부** (CBOE) |

### 2.2. 유료 데이터 (필요 시)

미국 시장에 본격적으로 가면 결국 사야 함:
- **Polygon.io** — 분단위 OHLCV, 실시간, 월 $29~199
- **IEX Cloud** — 저렴, 월 $9~
- **Alpha Vantage** — 프리티어 있음
- **RavenPack / Refinitiv News Analytics** — 뉴스 sentiment 정형화 (월 수천 달러+, 사실상 기관용)

→ 1,000만원 시드에 RavenPack 비용은 비합리적. **무료 + 자체 LLM 분류로 가야 함.**

### 2.3. 미국 시장의 "수급" 대체재

한국의 외인/기관 일별 데이터가 없는 대신:
- **13F 분기 공시** — 헤지펀드 보유 종목 (3개월 지연)
- **Form 4** — 내부자 매매 (실시간)
- **Short Interest** — 격주 공시
- **Dark Pool 데이터** — 일부 유료
- **Options flow** — Unusual Whales 등 ($30~50/월)

수급 시그널이 한국보다 **느리고 간접적**. 본인 시스템 설계의 핵심이 수급이라면 미국은 약점이다.

## 3. LLM 역할의 변화

### 3.1. 기회

- **Earnings call 트랜스크립트** — 한국에 거의 없는 데이터. LLM이 톤·뉘앙스 추출 가능
- **Reddit r/wallstreetbets, StockTwits** — 미국은 SNS 거래 심리 데이터 풍부
- **8-K 즉시 공시** — 정형화된 fact format. LLM이 분류 잘 함
- **Analyst report tone** — 정량 + 정성 결합 가능

### 3.2. 어려움

- **이미 다 한다.** Lopez-Lira & Tang 2023 ChatGPT 실험, Hybrid Sentiment 2025 등 미국 데이터로 무수히 검증됨
- **알파 부패가 빠름** — 미국은 발견 즉시 다들 따라옴
- **본인 시스템의 차별점이 안 나옴**

### 3.3. LLM 활용도가 더 높아지는 자리

미국 가면 LLM이 **주연**이 되어야 한다. 이유:
- 정량 시그널은 이미 다 짜낸 시장
- LLM이 처리할 텍스트 데이터는 풍부 (영어, earnings call, 분석가 리포트)
- 한국처럼 "수급 + 차트"로 가는 게 통하지 않음

## 4. 매매 측면 변화

### 4.1. 거래 시간 — 가장 현실적인 문제

미국 시장: 한국시간 **23:30~06:00** (서머타임 22:30~05:00)

**스윙은 OK** — 종가 기준 매매하면 큰 문제 없음. 한국 새벽~아침에 주문하면 됨.

**데이트레이딩은 ❌** — 본인 일상 파괴. 시도하지 마라.

### 4.2. 수수료 / 세금

수수료:
- Robinhood, Interactive Brokers — 0% 또는 매우 저렴
- 한국 증권사 해외주식 — 0.07~0.25% + 환전 스프레드

세금 (한국 거주자):
- 양도차익 22% (250만원 공제 후)
- 배당 15% 원천징수
- **잦은 매매 = 세금 폭탄** — 스윙으로 1년 안에 자주 회전시키면 수익의 22%가 세금
- 한국 주식보다 세금 부담이 **훨씬 큼**

### 4.3. 슬리피지 / 유동성

미국: 압도적으로 좋음. 1,000만원 시드는 시장에 영향 0.

한국: KOSDAQ 소형주는 1,000만원으로도 호가 밀림.

**미국의 장점.** 단, 위 세금 부담이 이를 상쇄.

### 4.4. 증권사 API

| 옵션 | 특징 |
|---|---|
| **Interactive Brokers (IBKR)** | 가장 표준. Python 라이브러리 (`ib_insync`) 잘 정리. 추천 |
| **Alpaca** | 무료, REST API 깔끔, 미국 거주자 한정 |
| **TD Ameritrade** | Schwab 인수로 API 정책 변화 중 |
| **한국 증권사 해외주식 API** | KIS 해외주식 지원 ⭕. 환전 자동 처리 ⭕ |

**한국 거주자 → KIS 해외주식 API가 합리적.** 환전·세금 자동 처리. IBKR은 환전·세금 직접 처리 필요.

## 5. 차별화 전략 — 한국 vs 미국

### 5.1. 한국 시장 차별점 (메인 지침서 §6 참고)

✅ 한국어 NLP / 외인·기관 수급 / 유튜브 시황 / 작은 시드

### 5.2. 미국 시장에서 가능한 차별점

미국에서 개인이 가질 수 있는 edge는 매우 한정적. 그래도 후보:

- **소형주 (Russell 2000 미만)** — 헤지펀드가 못 다루는 사이즈. 알파 가능성 ⭕. 그러나 유동성 함정 주의
- **이벤트 드리븐 + 즉시 반응** — 8-K 공시 직후 분 단위 반응. 인프라 비용 큼
- **틈새 섹터** — 본인이 도메인 지식 있는 산업 (예: 게임, 반도체)
- **Cross-market arbitrage** — 한국 ADR(쿠팡, NU 등)와 본주 차이. 매우 좁아진 영역

위 모두 한국 시장 차별화보다 **어렵다.**

### 5.3. 솔직한 권유

**한국 먼저, 미국은 1년 후 옵션.**

이유:
1. 한국에서 작동하는 시스템을 만든 다음에 → 그 인프라를 미국으로 확장 (비용 1/3)
2. 한국에서 안 되는 시스템은 미국에서도 안 된다
3. 1,000만원 시드 → 한국이 더 효율적
4. 일상·시간대 부담 없음

**미국 먼저 가는 경우:** 본인이 영어권 헤지펀드와 정면 승부할 자신이 있고, 시드가 5억+ 있고, 풀타임으로 할 수 있을 때.

## 6. 만약 그래도 미국으로 간다면

### 6.1. 데이터 파이프라인 변경

```python
# 한국:
from pykrx import stock
df = stock.get_market_ohlcv("20240101", "20241231", "005930")

# 미국:
import yfinance as yf
df = yf.download("AAPL", start="2024-01-01", end="2024-12-31")
```

### 6.2. Universe filter 재정의

본인의 한국 조건 (시총 1,000억~3조, 등락 5% 등)은 미국 시장에서 **다른 의미**. 재검증 필요.

미국 스윙에서 자주 쓰는 조건:
- Russell 2000 또는 S&P 600 종목
- 평균 거래량 100만주+
- ATR 5%+ (변동성)
- 200일 이동평균 위
- IBD 89+ (Investor's Business Daily) — 모멘텀 점수

**한국 조건 그대로 옮기지 말 것.** 시장 구조가 다름.

### 6.3. LLM 활용도 재설계

신호 비중을 한국 시스템과 다르게:

| | 한국 | 미국 |
|---|---|---|
| 수급 (외인·기관) | 40% | 10% (13F만) |
| 가격·거래량 | 30% | 30% |
| LLM 신호 (뉴스·공시) | 20% | **40%** |
| LLM 신호 (earnings call·SNS) | 10% | **20%** |

미국에서는 LLM이 더 큰 비중을 차지해야 한다.

### 6.4. 세금 계획

스윙에서 미국 주식은 **장기보유(1년 이상)** 전략 비중을 일부 섞는 게 나을 수도. 한국보다 더 신중한 회전 관리 필요.

## 7. 결론

미국 시장은 **언제든 갈 수 있는 옵션**으로 두되, **메인은 한국이 맞다.**

본인 프로젝트의 핵심 가설 — *"한국어 텍스트 + 한국 수급 + 한국 유튜브 시황을 LLM으로 통합"* — 은 한국에서만 의미 있다. 미국에서는 정면 경쟁이고, 본인 edge가 거의 사라진다.

**6주 로드맵은 한국 시장 기준 그대로 진행.** 미국 옵션은 12개월 후 검증된 시스템 위에 추가 모듈로 검토.
